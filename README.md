# Portifólio

Bem-vindo ao meu portfólio pessoal! Este é um site moderno e responsivo desenvolvido com tecnologias de ponta para apresentar meus projetos, habilidades e informações de contato.

## 🚀 Sobre o Projeto

Este portfólio foi criado para destacar minha trajetória como estudante de Engenharia de Software na FIAP, minhas experiências em desenvolvimento web, e meus projetos pessoais. O design é clean, com animações suaves e foco na experiência do usuário.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + Shadcn/ui
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Deploy**: GitHub Pages

## 📋 Funcionalidades

- **Seção Sobre**: Informações sobre minha formação e experiência
- **Projetos**: Showcase dos meus trabalhos desenvolvidos
- **Habilidades**: Tecnologias e ferramentas que domino
- **Contato**: Formas de entrar em contato (email, WhatsApp, LinkedIn, GitHub)
- **Design Responsivo**: Otimizado para desktop e mobile
- **Tema Dark/Light**: Interface moderna com gradientes e efeitos visuais

## 🏃‍♂️ Como Executar Localmente

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou yarn
